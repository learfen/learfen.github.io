¿Qué es la Programación?

La programación incluye el diseño del algoritmo, el codigo y las operaciones que se debe realizar para resolver un problema.

Conceptos clave: Lenguajes de programación, Algoritmos, Código fuente.

Algoritmos

Un algoritmo es una serie de pasos consecutivos y finitos para resolver un problema o realizar una tarea.

Conceptos clave: Eficiencia, Lógica.

Variables var a = 10

Una variable es un espacio en la memoria de la computadora donde se puede almacenar un valor que puede cambiar.

Un texto let nombre = "Ana"; Un numero let edad = 12; Una lista let edadHermanos = [13, 14]; Un booleano let tieneHermanos = true;

Conceptos clave: Tipos de datos: números, cadenas, booleanos.

Condicionales a==10

Las estructuras condicionales permiten tomar decisiones en base a condiciones que tienen solo 2 respuestas posibles si o no.

[ if ] si tiene hermanos entonces if (tieneHermanos) {
  console.log("Tiene hermanos");
} [ if / else ] si tiene hermanos entonces , sino if (tieneHermanos) {
  console.log("Tiene hermanos");
} else {
  console.log("No tiene hermanos");
}

Conceptos clave: condición, if, else.

Operadores

Hay 2 tipos basicos de operadores en las condiciones.

Lógicos ( y:&& ) (o:||) if (tieneAuto && tieneLicencia) {
  console.log("Puede manejar");
} Aritmeticos <, >, ==, !=, <=, >=
  • Mayor-menor
    • if (hora < 7 || hora > 12) {
      console.log("Escuela cerrada");
    }
  • Igualdad y distinto
    • if (hora == 12 || hora != 13) {
      console.log("Escuela cerrada");
    }

    Conceptos clave: operadores, y, o, igual, distinto, mayor, menor.

    Funciones

    Una función es un bloque de código realiza una tarea específica con nombre, esto permite que pueda reutilizarse.

    function saludar(nombre) {
      return "Hola, " + nombre;
    }

    Conceptos clave: Parámetros, Retorno de valor.

    HTML (HyperText Markup Language)

    HTML es el lenguaje de marcado utilizado para estructurar y presentar contenido en la web.

    <h1>¡Hola, Mundo!</h1>
    <p>Este es un párrafo.</p>

    Conceptos clave: Etiquetas, Estructura del documento.

    CSS (Cascading Style Sheets)

    CSS es el lenguaje que se utiliza para describir la presentación visual de una página web.

    h1 {
      color: blue;
    }

    Conceptos clave: Selectores, Estilos.

    JavaScript

    JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web.

    document.querySelector('h1').textContent = 'Hola, Mundo!';

    Conceptos clave: Manipulación del DOM, Eventos.

    Diagrama de flujo

    Un diagrama de flujo es una representación visual de un flujo de datos.

    Conceptos clave: Diagrama de flujo, sentido, operacion, condición, bucle.
